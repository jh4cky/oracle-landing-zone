## Project Name
OCI UI Password Module

## Description of the module
This module creates a password for an user in OCI.

## Requirements

| Name | Version |
|------|---------|
| terraform | >= 1.0.0 |
| oci | >= 4.67.3 |

## Modules

No modules.

## Inputs

| Name | Description | Type | Default | Required |
|------|-------------|------|---------|:--------:|
| user\_id | The OCID of the user. | `string` | n/a | yes |

## Outputs

| Name | Description |
|------|-------------|
| password | n/a |
